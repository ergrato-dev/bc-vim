# 💪 Prácticas - Semana 08

## 🎯 Objetivo

Convertir Neovim en un IDE completo con LSP, autocompletado, Treesitter, debugging y snippets.

---

## 📋 Lista de Ejercicios

| # | Ejercicio | Duración | Tema |
|---|-----------|----------|------|
| 1 | [LSP y Mason](01-ejercicio-lsp/) | 1h | mason, lspconfig, diagnósticos |
| 2 | [Autocompletado](02-ejercicio-cmp/) | 1h | nvim-cmp, fuentes, íconos |
| 3 | [Treesitter](03-ejercicio-treesitter/) | 45 min | Parsers, text objects, folding |
| 4 | [DAP y Snippets](04-ejercicio-dap-snippets/) | 1h | Debugger, snippets, terminal |

**Tiempo total estimado:** 3 horas 45 minutos

---

## ✅ Criterios de Completitud

- [ ] LSP funcionando con diagnóstico en tiempo real
- [ ] Autocompletado con 4 fuentes (LSP, buffer, path, snippets)
- [ ] Treesitter con text objects funcionales
- [ ] Debugger configurado para al menos 1 lenguaje
- [ ] Snippets personalizados creados
