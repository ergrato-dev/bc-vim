## 📖 Lecturas Recomendadas

- [Practical Vim (2nd Edition)](https://pragprog.com/titles/dnvim2/) — Capítulos 2, 8-12 (Dot Formula, Visual Mode, Command-line Mode)
- [Learn Vim (the Smart Way)](https://github.com/iggredible/Learn-Vim) — Capítulos 7-9 (Sustituciones, Comandos globales, Macros básicos)
- [Vim Help: change.txt](https://vimhelp.org/change.txt.html) — Documentación oficial de cambios, `.`, `:s`, `:g`
- [Vim Help: visual.txt](https://vimhelp.org/visual.txt.html) — Documentación oficial de Visual mode
- [Vim Regex Reference](https://vimhelp.org/pattern.txt.html) — Referencia completa de expresiones regulares
