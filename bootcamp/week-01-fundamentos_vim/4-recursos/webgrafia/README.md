## Referencia rápida

### Vim Cheat Sheet (rtorr.com)
- **URL**: https://vim.rtorr.com/
- **Uso**: Referencia visual de comandos. Guárdala en favoritos.
- **Esta semana**: Las secciones "Cursor movement", "Insert mode", "Editing", "Saving and Exiting".

### vimhelp.org
- **URL**: https://vimhelp.org/
- **Uso**: Documentación oficial de Vim en HTML (más legible que el `:help` del terminal).
- **Comandos de esta semana**: `:help hjkl`, `:help motion`, `:help inserting`, `:help usr_01`

### Neovim Documentation
- **URL**: https://neovim.io/doc/user/
- **Uso**: Equivalente para Neovim. Misma información, actualizada con features modernas.

## Artículos recomendados

### "Your problem with Vim is that you don't grok vi" (Stack Overflow)
- **URL**: https://stackoverflow.com/a/1220118
- **Por qué**: La respuesta más votada explica la filosofía modal de Vim. Leer después de `01-por-que-vim.md`. Cambia la perspectiva.

### "Vim is not an IDE. It's a text editor." (vimways.org)
- **URL**: https://vimways.org/2018/death-by-a-thousand-files/
- **Por qué**: Contexto sobre cuándo y por qué usar Vim vs IDEs. Útil para entender el enfoque del bootcamp.

### "Seven habits of effective text editing" (Bram Moolenaar)
- **URL**: https://www.moolenaar.net/habits.html
- **Por qué**: Escrito por el creador de Vim. Principios de edición eficiente que aplican a toda la carrera.

## Herramientas interactivas

### Vim Adventures
- **URL**: https://vim-adventures.com
- **Qué es**: Juego en el navegador para aprender hjkl y movimientos básicos. Las primeras 3 pantallas son gratuitas.
- **Cuándo**: Buen complemento para los ejercicios de `2-practicas/01-ejercicio-vimtutor/`.

### OpenVim Interactive Tutorial
- **URL**: https://www.openvim.com/
- **Qué es**: Tutorial interactivo de Vim en el navegador. Sin instalación.
- **Cuándo**: Útil si quieres practicar antes de tener Neovim configurado.

### VimGolf
- **URL**: https://www.vimgolf.com
- **Qué es**: Desafíos donde el objetivo es realizar una edición con el mínimo de pulsaciones.
- **Esta semana**: Los desafíos más simples (rating "Easy"). No te frustres si no encuentras la solución óptima — el proceso ya enseña.
