# Videografía — Semana 04

Videos para interiorizar buffers, ventanas, splits, pestañas y netrw.

## Esenciales esta semana

### Vim Windows and Tabs Explained
- **URL**: https://www.youtube.com/watch?v=Wx3rOv7A8t8
- **Cuándo verlo**: Después de `02-ventanas-splits.md` y `04-pestanas.md`
- **Por qué**: Es la explicación visual más directa de la diferencia entre ventana y pestaña — la confusión más común de quien viene de VS Code o Sublime.

### How to Do 90% of What Plugins Do (With Just Vim)
- **URL**: https://www.youtube.com/watch?v=XA2WjJbmmoM
- **Cuándo verlo**: Después de completar toda la teoría de la semana
- **Por qué**: Cubre buffers y netrw como reemplazo nativo de plugins tipo NERDTree o file finders — refuerza que esta semana es sobre dejar de depender de plugins para lo básico.

### Netrw Guide (DevOnDuty)
- **URL**: https://www.youtube.com/watch?v=Qem8cpbJeYc
- **Cuándo verlo**: Después de la Parte 1 de `05-netrw-y-help.md`
- **Por qué**: Demostración práctica de crear, renombrar y eliminar archivos desde netrw sin salir de Vim.

### Vim As Your Editor (ThePrimeagen) — episodios de splits y buffers
- **URL**: https://www.youtube.com/playlist?list=PLm323Lc7iSW_wuxqmKx_xxNtJC_hJbQ7R
- **Cuándo verlo**: Como refuerzo después de `01-buffers.md` y `03-navegacion-ventanas.md`
- **Por qué**: Busca en la lista los episodios sobre gestión de buffers y mappings de `Ctrl-w` — muestra flujos reales de desarrollo con múltiples archivos abiertos.
