# 💪 Prácticas - Semana 02

## 🎯 Objetivo

Consolidar motions avanzados, text objects y el paradigma operador+sustantivo mediante ejercicios guiados con archivos reales de código.

---

## 📋 Lista de Ejercicios

| # | Ejercicio | Duración | Tema |
|---|-----------|----------|------|
| 1 | [Drill f/t](01-ejercicio-ft/) | 45 min | Búsqueda en línea con f, t, ;, , |
| 2 | [Código Anidado](02-ejercicio-text-objects/) | 1h | Text objects en JSON, HTML y código |

**Tiempo total estimado:** 3 horas 30 minutos

---

## 📖 Metodología

Cada ejercicio sigue el formato de **tutorial guiado**:

1. **Lee** las instrucciones en el `README.md` del ejercicio
2. **Genera o abre** el archivo de práctica proporcionado
3. **Ejecuta** los comandos en Vim (escribe, no copies y pegues)
4. **Verifica** que el resultado es el esperado
5. **Experimenta** variaciones por tu cuenta

> 💡 **Tip**: Esta semana el objetivo NO es terminar rápido. Es pensar en "frases Vim" (verbo + sustantivo) antes de ejecutar.

---

## ✅ Criterios de Completitud

Para considerar un ejercicio completado:

- [ ] Comandos ejecutados sin mirar referencia
- [ ] Usaste text objects y motions, no edición carácter por carácter
- [ ] Puedes explicar por qué usaste cada text object
- [ ] Formulaste cada acción como frase Vim (verbo + sustantivo)
