## 💡 Recomendación

> Los mappings de navegación (`Ctrl-hjkl` para splits, `<leader>bn/bp` para buffers) son la inversión más rentable de esta semana. Configúralos HOY. Cada día que pases sin ellos estás desperdiciando cientos de pulsaciones.
