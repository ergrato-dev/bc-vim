# Ebooks Gratuitos — Semana 01

Recursos de lectura sin costo para los fundamentos de Vim.

---

## Imprescindibles esta semana

### Learn Vim (the Smart Way)
- **Autor**: Igor Irianto (iggredible)
- **URL**: https://github.com/iggredible/Learn-Vim
- **Capítulos recomendados**: Ch. 1 (Starting Vim), Ch. 2 (Buffers, Windows, and Tabs), Ch. 3 (Opening and Searching Files)
- **Por qué**: La guía más completa y actualizada de Vim, disponible gratis en GitHub. Explica la filosofía modal de forma clara.

### Vim User Manual (oficial)
- **Autor**: Bram Moolenaar
- **URL**: https://vimhelp.org/usr_toc.txt.html
- **Secciones recomendadas**: `usr_01.txt` (Introducción), `usr_02.txt` (Primeros pasos), `usr_03.txt` (Mover alrededor)
- **Acceso desde Vim**: `:help user-manual`
- **Por qué**: La documentación oficial. Accesible directamente desde el editor con `:help`.

### A Byte of Vim
- **Autor**: Swaroop C H
- **URL**: https://vim.swaroopch.com/
- **Capítulos recomendados**: Introduction, First Steps, Modes
- **Por qué**: Guía introductoria corta (~90 páginas) con ejemplos progresivos. Ideal para el primer contacto.

## Complementario

### Vim Reference Manual
- **URL**: https://vimhelp.org/
- **Uso**: Referencia técnica completa. No leer de corrido — consultarlo cuando tengas dudas sobre un comando específico.
- **Acceso desde Vim**: `:help <comando>` para cualquier comando
