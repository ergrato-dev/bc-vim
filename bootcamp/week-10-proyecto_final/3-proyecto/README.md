# 🎯 Proyecto Final: Tu Configuración de Neovim

## 📋 Descripción

Este es el proyecto integrador final del bootcamp. Construirás, documentarás y publicarás tu configuración personal de Neovim como un repositorio profesional en GitHub.

---

## 🎯 Objetivos

- ✅ Consolidar todos los conocimientos de las 9 semanas anteriores
- ✅ Construir una configuración de Neovim completa desde cero
- ✅ Publicar dotfiles en GitHub con documentación profesional
- ✅ Demostrar fluidez en edición con Vim en un proyecto real
- ✅ Presentar y defender tu configuración

---

## 📦 Requisitos

### Requisitos Mínimos (Aprobación 70%)

| # | Requisito | Semana relacionada |
|---|-----------|-------------------|
| 1 | Estructura modular: init.lua + lua/ con separación por responsabilidad | 7 |
| 2 | lazy.nvim con lazy-loading configurado | 6 |
| 3 | LSP: 2+ language servers con diagnóstico y autocompletado | 8 |
| 4 | Apariencia: tema + lualine + file tree | 6 |
| 5 | Git: gitsigns y/o fugitive | 6, 9 |
| 6 | Keymappings documentados con descripciones | 7 |
| 7 | README.md con documentación e instrucciones | 10 |

### Requisitos Avanzados (Excelencia 100%)

| # | Requisito |
|---|-----------|
| 8 | 10+ snippets personalizados para tu lenguaje principal |
| 9 | DAP: debugger configurado para tu lenguaje |
| 10 | Treesitter: text objects y folding avanzados |
| 11 | Tests: configuración para ejecutar tests desde Vim |
| 12 | Formateo/Linting: none-ls o conform.nvim + nvim-lint |
| 13 | Sesiones: restauración automática |
| 14 | 3+ autocmds útiles por filetype |
| 15 | Telescope: 5+ pickers con keybindings |
