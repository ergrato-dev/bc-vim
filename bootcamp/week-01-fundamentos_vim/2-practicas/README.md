# 💪 Prácticas - Semana 01

## 🎯 Objetivo

Consolidar los conceptos aprendidos en la teoría mediante ejercicios prácticos guiados dentro de Vim.

|---|-----------|----------|------|

**Tiempo total estimado:** 3 horas

---

## 📖 Metodología

Cada ejercicio sigue el formato de **tutorial guiado**:

1. **Lee** las instrucciones en el `README.md` del ejercicio
2. **Abre** el archivo de práctica proporcionado
3. **Ejecuta** los comandos en Vim (no copies y pegues, escribe)
4. **Verifica** que el resultado es el esperado
5. **Experimenta** variaciones por tu cuenta

> 💡 **Tip**: No copies y pegues los comandos. Escríbelos manualmente. La memoria muscular se construye tecleando.

---

## ✅ Criterios de Completitud

Para considerar un ejercicio completado:

- [ ] Código/comandos ejecutados sin errores
- [ ] Comprendes qué hace cada comando
- [ ] Puedes explicar el concepto a alguien más
- [ ] Has experimentado con variaciones
