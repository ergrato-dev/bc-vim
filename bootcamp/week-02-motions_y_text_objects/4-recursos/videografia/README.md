# Videografía — Semana 02

Videos para interiorizar motions, text objects y la gramática de Vim.

---

## Esenciales esta semana

### Mastering the Vim Language (Chris Toomey) — completo
- **URL**: https://www.youtube.com/watch?v=wlR5gYd6um0
- **Duración**: ~40 min
- **Cuándo verlo**: Después de leer `05-vim-como-lenguaje.md`
- **Por qué**: El video más importante de toda la semana. Explica el paradigma verbo+sustantivo mejor que cualquier texto. Cambia cómo piensas Vim para siempre.

### Vim Text Objects — The Definitive Guide (ThePrimeagen)
- **URL**: https://www.youtube.com/watch?v=JnD9Uro_oqc
- **Duración**: ~15 min
- **Cuándo verlo**: Después de leer `04-text-objects.md`
- **Por qué**: Demostración en código real de `ci"`, `da{`, `yit` y combinaciones con `.`. Muy práctico.

### Vim Search and Replace Tutorial
- **URL**: https://www.youtube.com/watch?v=PzmLJy0o6qo
- **Duración**: ~12 min
- **Cuándo verlo**: Después de `02-busqueda-en-archivo.md`
- **Por qué**: Muestra `/` y `?` con `smartcase`, `hlsearch`, `incsearch` en acción. Útil para confirmar la configuración.
