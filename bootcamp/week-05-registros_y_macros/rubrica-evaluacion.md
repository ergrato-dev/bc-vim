# Rúbrica de Evaluación - Semana 5: Registros y Macros

## Evidencia de Conocimiento (30%)

| Criterio | Excelente (100%) | Bueno (80%) | Suficiente (60%) | Insuficiente (0%) |
|----------|------------------|-------------|-------------------|--------------------|
| Tipos de registros | Diferencia "", "0, "a-"z, "+, "* con ejemplos | Diferencia la mayoría | Solo conoce "" | No los conoce |
| Macros recursivas | Explica el mecanismo y cuándo usarlas | Sabe que existen | Ha oído de ellas | No las conoce |
| "+ vs "* | Explica la diferencia en Linux (clipboard vs primary) | Describe uno correctamente | Los confunde | No los conoce |

## Evidencia de Desempeño (40%)

| Criterio | Excelente (100%) | Bueno (80%) | Suficiente (60%) | Insuficiente (0%) |
|----------|------------------|-------------|-------------------|--------------------|
| Macro para procesar líneas | Macro funciona en todas las líneas, sin errores | Funciona en >90% líneas | Funciona en >70% | No funciona |
| Aplicación batch | Usa :normal @q o número+@q correctamente | Usa batch con error menor | Solo aplica manual línea por línea | No aplica batch |
| Corrección de errores | Identifica y corrige errores en la macro via :reg | Corrige con ayuda | No puede corregir | No intenta corregir |

## Evidencia de Producto (30%)

| Criterio | Excelente (100%) | Bueno (80%) | Suficiente (60%) | Insuficiente (0%) |
|----------|------------------|-------------|-------------------|--------------------|
| Conversión logs → JSON | 100% correcto, JSON válido | JSON válido con errores de formato | JSON parcial | No entregado |
| Uso de registros con nombre | Usa registros a-d para almacenar patrones | Usa al menos 2 registros | Usa 1 registro | No usa registros con nombre |
| Documentación de macros | Cada macro documentada con propósito y comandos | Macros documentadas sin detalle | Solo lista comandos sueltos | Sin documentación |
