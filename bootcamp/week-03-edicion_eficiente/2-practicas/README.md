# 💪 Prácticas - Semana 03

## 🎯 Objetivo

Consolidar el comando `.`, Visual mode, marcas, sustituciones y comandos shell mediante ejercicios guiados con archivos de datos y código real.

---

## 📋 Lista de Ejercicios

| # | Ejercicio | Duración | Tema |
|---|-----------|----------|------|
| 1 | [Dot Command Drill](01-ejercicio-dot/) | 45 min | Repetición con `.` + `n` |
| 2 | [Visual Block](02-ejercicio-visual-block/) | 1h | Ediciones rectangulares con Ctrl-v |
| 3 | [Marcas](03-ejercicio-marcas/) | 45 min | Navegación con marcas locales y globales |
| 4 | [Sustituciones y Shell](04-ejercicio-sustituciones/) | 1h | :s, :%s, :g, :r !, !{motion} |

**Tiempo total estimado:** 3 horas 30 minutos

---

## 📖 Metodología

Cada ejercicio sigue el formato de **tutorial guiado**:

1. **Lee** las instrucciones en el `README.md` del ejercicio
2. **Genera o abre** el archivo de práctica proporcionado
3. **Ejecuta** los comandos en Vim (escribe, no copies y pegues)
4. **Verifica** que el resultado es el esperado
5. **Experimenta** variaciones por tu cuenta

> 💡 **Tip**: El comando `.` es el tema central de esta semana. Si al final del día no estás usando `.` instintivamente, repite el ejercicio 1.

---

## ✅ Criterios de Completitud

Para considerar un ejercicio completado:

- [ ] Comandos ejecutados sin mirar referencia
- [ ] Usaste `.` para repetir cambios en lugar de rehacerlos manualmente
- [ ] Usaste `Ctrl-v` + `I`/`A` para editar columnas
- [ ] Usaste `:s` y `:%s` con confianza
- [ ] Usaste marcas para navegar entre secciones
