# Ebooks Gratuitos — Semana 03

Dot command, Visual mode, marcas, sustituciones y filtros de shell.

---

## Imprescindibles esta semana

### Practical Vim (2nd Edition) — Capítulos 8-12
- **Autor**: Drew Neil
- **URL**: https://pragprog.com/titles/dnvim2/
- **Capítulos recomendados**:
  - **Ch. 8 (Meet the Dot Formula)** — el capítulo más importante del libro
  - Ch. 9 (Visual Mode)
  - Ch. 10 (Ex Command Line)
  - Ch. 11 (Substitution)
  - Ch. 12 (Global Command)
- **Por qué**: Es EL libro de referencia para esta semana. Cada tip está construido alrededor de hacer los cambios repetibles con `.`.

### Learn Vim (the Smart Way) — Capítulos 7-9, 13
- **Autor**: Igor Irianto (iggredible)
- **URL**: https://github.com/iggredible/Learn-Vim
- **Capítulos recomendados**:
  - Ch. 7 (Marks) — marcas locales y globales a fondo
  - **Ch. 12 (Search and Substitute)** — `:s`, `:g`, flags de sustitución
  - Ch. 13 (Undo) — cómo interactúa `.` con `u` y `Ctrl-r`
- **Por qué**: Complementa Practical Vim con ejemplos actualizados y capítulo dedicado exclusivamente a marcas.

### Vim User Manual — Ex Commands
- **Autor**: Bram Moolenaar
- **URL**: https://vimhelp.org/usr_toc.txt.html
- **Secciones recomendadas**: `usr_10.txt` (Basic editing), `usr_12.txt` (Clever tricks), `usr_25.txt` (Editing files)
- **Acceso desde Vim**: `:help usr_12`
- **Por qué**: Cubre `:s`, `:g` y filtros con `!` desde la fuente oficial.

## Complementario

### `:help change.txt`
- **URL**: https://vimhelp.org/change.txt.html
- **Por qué**: Documentación técnica completa del comando `.`, operadores y sustituciones. Referencia cuando `:s` no hace lo esperado.

### `:help visual.txt`
- **URL**: https://vimhelp.org/visual.txt.html
- **Por qué**: Documentación oficial de Visual mode, incluyendo comandos poco conocidos como `gv`, `o`, `O` y block-insert (`I`/`A` en Visual block).
