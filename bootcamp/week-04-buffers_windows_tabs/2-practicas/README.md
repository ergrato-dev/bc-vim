# 💪 Prácticas - Semana 04

## 🎯 Objetivo

Consolidar la gestión de buffers, ventanas, pestañas, netrw y el sistema de ayuda mediante ejercicios guiados con archivos reales.

---

## 📋 Lista de Ejercicios

| # | Ejercicio | Duración | Tema |
|---|-----------|----------|------|
| 1 | [Buffer Management](01-ejercicio-buffers/) | 45 min | :ls, :b, :bd, :bn, :bp, Ctrl-6 |
| 2 | [Split Navigation](02-ejercicio-splits/) | 1h | Splits, navegación, redimensión |
| 3 | [Tab Workflow](03-ejercicio-pestanas/) | 45 min | Pestañas como workspaces |
| 4 | [netrw y :help](04-ejercicio-netrw-help/) | 1h | Explorador de archivos + ayuda |

**Tiempo total estimado:** 3 horas 30 minutos

---

## 📖 Metodología

Cada ejercicio sigue el formato de **tutorial guiado**:

1. **Lee** las instrucciones
2. **Ejecuta** los comandos en Vim
3. **Verifica** que el resultado es el esperado
4. **Experimenta** variaciones

> 💡 **Tip**: Esta semana es sobre el "entorno de trabajo". No memorices comandos — practica workflows completos hasta que sean naturales.

---

## ✅ Criterios de Completitud

- [ ] Gestionas buffers sin pensar (abrir, listar, cambiar, eliminar)
- [ ] Creas y navegas splits con fluidez
- [ ] Usas pestañas como layouts, no como archivos individuales
- [ ] Navegas netrw y gestionas archivos desde él
- [ ] Consultas `:help` sin salir de Vim
