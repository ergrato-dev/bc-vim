# 🎓 Ejercicio 01: vimtutor

## 🎯 Objetivo

Completar el tutorial interactivo oficial de Vim.

**Duración estimada:** 1 hora

---

## 📋 Requisitos Previos

- Vim o Neovim instalado
- 1 hora sin interrupciones

---

## 📝 Instrucciones

### Paso 1: Abrir vimtutor

```bash
# Para Neovim:
nvim +Tutor

# Para Vim clásico:
vimtutor
```

### Paso 2: Seguir el Tutorial

vimtutor tiene 7 lecciones. Cada lección te enseña un concepto y te pide que practiques.

| Lección | Contenido | Tiempo |
|---------|-----------|--------|
| 1.1-1.3 | Movimiento básico (hjkl) | 10 min |
| 1.4-1.7 | Edición básica (x, i, a) | 10 min |
| 2.1-2.3 | Eliminar (dw, dd) | 5 min |
| 2.4-2.7 | Deshacer y más comandos | 10 min |
| 3.1-3.3 | Pegar y reemplazar (p, r) | 5 min |
| 4.1-4.3 | Cambiar texto (cw, c$) | 10 min |
| 5.1-5.4 | Archivos y búsqueda | 10 min |

### Paso 3: Verificar

Al terminar vimtutor deberías poder:

- [ ] Moverte con hjkl sin pensar
- [ ] Insertar texto con i, a, A
- [ ] Eliminar palabras y líneas con dw, dd
- [ ] Deshacer con u y rehacer con Ctrl-r
- [ ] Copiar y pegar con yy, p
- [ ] Buscar con / y navegar con n, N
- [ ] Guardar y salir con :wq

### Paso 4: Repetir (Opcional)

Si algo no quedó claro, vimtutor guarda tu progreso. Puedes retomarlo o empezar de nuevo:

```bash
nvim +Tutor
```

---

## ✅ Checklist de Verificación

- [ ] Completé las 7 lecciones de vimtutor
- [ ] Entiendo cada comando enseñado
- [ ] Puedo ejecutar los comandos sin mirar una referencia
