## Complementario

### Búsquedas recomendadas en YouTube esta semana

En lugar de enlaces puntuales que pueden quedar desactualizados, busca estos títulos exactos — son términos estables usados por varios creadores de contenido de Vim:

- **"vim dot command tutorial"** — refuerza `01-dot-command.md` con ejemplos en código real
- **"vim visual block mode"** — Ctrl-v para alinear columnas, útil antes del proyecto semanal (CSV → markdown)
- **"vim marks tutorial"** — demostraciones de `ma`, `` `a ``, `'a` navegando archivos grandes
- **"vim global command tutorial"** o **"vim :g command"** — `:g/patrón/comando` en acción, el complemento natural de `.`
- **"vim filter external command"** o **"vim bang sort"** — el operador `!` con `sort`, `column -t`, `jq`

> **Estrategia**: prioriza canales que ya conoces de semanas anteriores (ThePrimeagen, DevOnDuty, Thoughtbot) — mantienen el mismo nivel de profundidad técnica.
