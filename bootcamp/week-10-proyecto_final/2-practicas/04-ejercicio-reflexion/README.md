# 📖 Ejercicio 04: Reflexión y Publicación

## 🎯 Objetivo

Escribir tu reflexión final, publicar el repositorio en GitHub y completar tu graduación.

**Duración estimada:** 1 hora

---

## 📝 Instrucciones

### Paso 1: Escribir la Reflexión

Crea `~/reflexion-bootcamp.md`:

```markdown
# Reflexión — Bootcamp Vim Zero to Hero

## ¿Qué fue lo más difícil de aprender?

[Tu respuesta honesta aquí. 3-5 párrafos.]

## ¿Qué es lo que más usas en tu día a día?

[Comandos, plugins, flujos que ya son parte de tu rutina.]

## ¿Qué te gustaría seguir explorando?

[Lo que te interesa aprender después del bootcamp.]
```

### Paso 2: Publicar en GitHub

```bash
cd ~/.config/nvim

# Verificar estado
git status

# Asegurar que lazy-lock.json está incluido
git add lazy-lock.json

# Agregar README y screenshots
git add README.md screenshots/

# Commit final
git commit -m "feat: configuración final del bootcamp"

# Push
git push origin main
```

### Paso 3: Verificar el Repositorio

```text
1. Abre https://github.com/tu-usuario/nvim
2. Verifica que README se ve bien
3. Verifica que screenshots cargan
4. Verifica que lazy-lock.json está presente
5. Copia la URL para la entrega
```

### Paso 4: Completar Entregables

```text
☐ URL del repositorio GitHub: ________________________________
☐ Video demo / screencast (si requerido): ________________________________
☐ Documento de reflexión: ________________________________
```

### Paso 5: Cápsula del Tiempo

Guarda para tu yo futuro:

```text
Hoy, [fecha], completé el Bootcamp Vim Zero to Hero.

Mi comando favorito: ________________________________
Mi plugin favorito: ________________________________
Lo que más me costó: ________________________________
Lo que más me sorprendió: ________________________________

En 6 meses volveré a leer esto.
```
