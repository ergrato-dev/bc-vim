# Videografía — Semana 01

Videos para reforzar los fundamentos de Vim visualmente.

---

## Esenciales esta semana

### Vim As Your Editor — Introducción (ThePrimeagen)
- **URL**: https://www.youtube.com/watch?v=H3o4l4GVLW0
- **Duración**: ~29 min
- **Cuándo verlo**: Antes o durante la lectura de `01-por-que-vim.md`
- **Por qué**: Motivación real de un desarrollador senior. Muestra por qué Vim sigue siendo relevante en 2024.

### Vim Basics in 8 Minutes (Tutorialzine)
- **URL**: https://www.youtube.com/watch?v=ggSyF1SVFr4
- **Duración**: ~8 min
- **Cuándo verlo**: Después de leer `02-modos-de-vim.md`
- **Por qué**: Demostración rápida de modos, movimiento y edición básica. Refuerza la transición entre modos visualmente.

### Mastering the Vim Language (Chris Toomey)
- **URL**: https://www.youtube.com/watch?v=wlR5gYd6um0
- **Duración**: ~40 min (puedes ver los primeros 20 min esta semana)
- **Cuándo verlo**: Después de `03-movimiento-basico.md`
- **Por qué**: Explica el paradigma "verbo + sustantivo" mejor que cualquier texto. Seminal.
