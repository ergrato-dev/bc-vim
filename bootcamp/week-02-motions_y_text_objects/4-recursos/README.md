## 📖 Lecturas Recomendadas

- [Practical Vim (2nd Edition)](https://pragprog.com/titles/dnvim2/) — Capítulos 2-4 (Normal Mode), 5-7 (Visual Mode), 8-9 (Command-line Mode)

- [Vim Help: motion.txt](https://vimhelp.org/motion.txt.html) — Documentación oficial de todos los motions y text objects
- [Vim Text Objects: The Definitive Guide](https://blog.carbonfive.com/vim-text-objects-the-definitive-guide/) — Artículo canónico sobre text objects
