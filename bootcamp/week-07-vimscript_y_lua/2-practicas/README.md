# 💪 Prácticas - Semana 07

## 🎯 Objetivo

Dominar la API Lua de Neovim: opciones, keymappings, autocmds, comandos de usuario y Vimscript básico.

---

## 📋 Lista de Ejercicios

| # | Ejercicio | Duración | Tema |
|---|-----------|----------|------|
| 1 | [API Lua y Keymaps](01-ejercicio-api-keymaps/) | 1h | vim.opt, vim.keymap, vim.cmd, vim.fn |
| 2 | [Autocmds](02-ejercicio-autocmds/) | 1h | nvim_create_autocmd, augroup |
| 3 | [Comandos de Usuario](03-ejercicio-comandos/) | 45 min | nvim_create_user_command, módulos |
| 4 | [Vimscript Legacy](04-ejercicio-vimscript/) | 30 min | Lectura y conversión Vimscript→Lua |

**Tiempo total estimado:** 3 horas 15 minutos

---

## ✅ Criterios de Completitud

- [ ] Configuras opciones con `vim.opt`, `vim.g`, `vim.bo`, `vim.wo`
- [ ] Creas keymaps con `vim.keymap.set` y descripciones
- [ ] Creas autocmds con `augroup` y `{ clear = true }`
- [ ] Creas comandos de usuario con argumentos y completado
- [ ] Lees y conviertes Vimscript a Lua
